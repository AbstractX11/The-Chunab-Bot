# The Chunab Bot

The Chunab Bot is a twitter bot coded in python that sends live instant vote count updates of "Parliament & Provincial Election Nepal 2079"

# How to get the bot running

Follow twitter documentation to create a developer account and get the required keys from there

> Setting up Environmental Variables

Create .env file in root directory of the project and paste your keys there.
Note: You can install **python-dotenv** in your virtual environment using **pip**

> Run this in command line to install python-dotenv
```
pip install python-dotenv

```

> This is how your .env file should look like
```
APIKEY = your api key
APISECRETKEY = your api secret key
ACCESSKEY = your access key
ACCESSKEYSECRET =your access key secret
```
